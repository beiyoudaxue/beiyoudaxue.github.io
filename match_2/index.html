匹配表单（第二版）

内容

本表单共有4个大类：

  1. 个人信息
  2. 兴趣类
  3. 个人趋向类
  4. 性格类

个人信息

在第一项：个人信息，只有当：

  1. 对方满足你填写的要求
  2. 你满足对方填写的要求

才会进行之后的匹配，否则不会进行匹配。

表单的目的就是 实现 双份满足双方的要求

电子邮件地址

建议填写QQ邮箱。

学号

选择学号的后三位即可

身高 + 体重

表单内容这一项该为：身高 + BMI(身体质量指数)

自己老家经纬度：

经纬度查询地址http://jingweidu.757dy.com

想找一个离家近一些的，距离多远，自己决定

现居住经纬度：

经纬度查询地址http://jingweidu.757dy.com

不想异地，你可以选择近一些

兴趣类

  1. 兴趣类共有若干选项，自己的信息可以 和 要求对方的信息 可以填写的不一致。
  2. 该项采取百分制，50%为周围人的平均兴趣水平，值越高 对则项越喜爱。
  3. 权重越大，则这项在计算分数中，占比越高。
  4. 禁止所有选项全填高分，否则认为是“不健康”的数据，不进行匹配。

个人趋向

  1. 个人趋向类选项共有若干个选项，至少填写一项，也可以全部填写。

  2. 自己的信息可以和要求对方的信息不一致。

  3. 该项采取百分制，50%为周围人的平均兴趣水平，值越高 对则项越喜爱。

  4. 权重越大，则这项在计算分数中，占比越高。

  5. 禁止所有选项全填同样的数据，否则认为是“不健康”的数据，不进行匹配。

性格类

  1. 性格类选项共有若干选项，至少填写一项，也可以全部填写。
  2. 自己的信息可以和要求对方的信息不一致。
  3. 该项采取百分制，50%为周围人的平均值。
  4. 权重越大，则这项在计算分数中，占比越高。
  5. 禁止所有选项全填同样的数据，否则认为是“不健康”的数据，不进行匹配。

匹配公式

双方每匹配一次，每个人都有一个得分：


$$
分数(百分制) = 100分 * \sum_{所有项的和}^{} 相对权重 * \left | 自己要求的数据 - 对方填写的数据 \right | * 函数 * 双方该项权重都不为0
$$
其中：

  1. 相对权重: 相对权重 = 该项的权重 / 所有项的权重之和
  2. 自己要求的数据： 自己表单中的第三列
  3. 对方填写的数据： 对方表单中的第二列
  4. 函数：从图中可以看到，自己要求的数据 与 对方填写的数据 越接近，则得分越高。

双方该项权重都不为0： 该项中，任何一方的权重为0，则认为他对该项没有兴趣，则该项得分为0。

最终双方的匹配值为：


$$
匹配值 = (男生的分数 + 女生的分数) / 2 (百分制)
$$

值越大，则越匹配。

结果公布

系统会以邮箱方式发送匹配结果(匹配双方的邮箱 和 匹配值)，建议填写QQ邮箱。
